## HStack:
- horizontally stack all the elements that are present inside of this
```swift
 HStack () {
	 Text("Knock, Knock!")
		 .padding()
		 .background(Color.yellow)
		 .cornerRadius(20)
		 .shadow(radius: 20)
		
	 Text("Knock, Knock!")
		 .padding()
		 .background(Color.yellow)
		 .cornerRadius(20)
		 .shadow(radius: 20)
}
```
